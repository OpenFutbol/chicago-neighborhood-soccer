# Chicago Neighborhood Soccer

This repository is part of the [OpenFutbol](https://github.com/openfutbol) organization and serves as an archive of early Chicago Neighborhood Soccer (NBHD Soccer) community development.

## Current Resources

Chicago Neighborhood Soccer has moved to its own organization. For the latest information:

- **Website:** [nbhdsoccer.org](https://nbhdsoccer.org)
- **GitHub Organization:** [github.com/nbhdsoccer](https://github.com/nbhdsoccer)

## About

Chicago Neighborhood Soccer is a 501(c)(3) nonprofit and grassroots, community-driven organization for local pick-up soccer players in Chicago. The community celebrates pick-up soccer and gives back to local neighborhoods.

## Contact

For questions or to get involved, visit [nbhdsoccer.org](https://nbhdsoccer.org) or reach out via the main organization at [github.com/nbhdsoccer](https://github.com/nbhdsoccer).
